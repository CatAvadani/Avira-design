## Installation

1. **Clone the repository**

   ```bash
   git clone (https://github.com/CatAvadani/Avira-design.git)
   cd avira-design
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
